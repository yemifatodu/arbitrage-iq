# 🚀 Crypto Arbitrage Tracker Pro

Professional real-time cryptocurrency arbitrage detection across 7 major exchanges.

## ✨ Features

- **7 Exchanges**: Binance, KuCoin, Kraken, Coinbase, Bitfinex, Bybit
- **Real-time Monitoring**: Concurrent API calls for fast data
- **Smart Detection**: Calculates net profit after fees
- **Professional UI**: Dark theme with interactive dashboard
- **SQLite Database**: Historical tracking and analytics
- **Telegram Alerts**: Instant notifications for opportunities
- **Export CSV**: Download historical data
- **Risk Assessment**: Color-coded opportunity cards

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/yemifatodu/crypto-arbitrage-tracker.git
cd crypto-arbitrage-tracker

# Install dependencies

# Run application
